WebObs

Configuration files overview

This document describes configuration files from WebObs-alpha-1.0.3 release. The main configuration file WEBOBS.rc is automatically loaded by the WebObs::Configmodule, creating the %WEBOBS perl hash, to which is automatically appended the definitions from WEBOBS.conf file, representing legacy variables not yet ported to the new system.

The WebObs configuration folder is pointed to by the symbolic link /etc/webobs.d.
